

    <h1>Frontend Assignment: Convert Figma Design to HTML & CSS</h1>

    <p>
        This project is a responsive webpage created by converting the provided
        Figma design into semantic HTML5 and vanilla CSS. The layout is fully
        responsive and adapts to desktop, tablet, and mobile screen sizes.
    </p>

    <hr>

    <h2>How to Run the Project</h2>

    <ol>
        <li>Download or clone the project folder.</li>
        <li>Open the project in Visual Studio Code.</li>
        <li>Open <strong>index.html</strong> in your browser.</li>
        <li>Resize the browser window to test the responsive design.</li>
