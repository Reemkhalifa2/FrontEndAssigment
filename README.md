## 📖 Project Overview

This project converts the provided Figma design into a fully responsive webpage using only **HTML5** and **CSS3**.

The webpage follows best practices by using semantic HTML, clean CSS, Flexbox/Grid layouts, CSS variables, and media queries for responsiveness.
